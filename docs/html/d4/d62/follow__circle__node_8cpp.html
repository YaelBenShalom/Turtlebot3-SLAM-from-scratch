Detailed Description

A node that publishes commands that let the robot drive in a circle of a specified radius at a specified speed.

PARAMETERS: frequency (int): control loop frequency control_flag (bool): specifies when new /control message is read turtle_speed (float): The linear speed of the robot turtle_controlled_speed (float): The linear speed of the robot, when taking into account the control service circle_radius (float): The radius of the circle twist (geometry_msgs::Twist): the robot's twist PUBLISHES: cmd_vel (geometry_msgs/Twist): Publishes to the robot's velocity. SERVICES: Control (control): Causes the robot to travel either clockwise, counter clockwise, or stop. 1 - clockwise, -1 - counter clockwise, 0 - stop.
